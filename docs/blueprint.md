# **App Name**: CondoConnect

## Core Features:

- Role Selection: Role selection screen for admin or owner login.
- User Authentication: Authentication forms for both admin and owner roles.
- Admin Dashboard: Admin dashboard with summaries of payments, pending dues, and total units.
- Owner Dashboard: Owner dashboard displaying last payment, next payment date, and unit details.
- Payment History: Display a list of the last 10 payments
- Smart Community Updates: Use a tool that decides what the top 3 most relevant community updates are for a user, based on their profile and payment history, for display in the user's dashboard.

## Style Guidelines:

- Primary color: Deep blue (#3066BE) to convey trust and professionalism, slightly less saturated than the project's original color. 
- Background color: Light blue-gray (#E8F0FE), a desaturated tint of the primary color, to provide a clean and calming backdrop.
- Accent color: Muted violet (#79589F), a desaturated, darker analogous hue to the primary, used for interactive elements.
- Body and headline font: 'Inter', a sans-serif font for clear, modern text.
- Font Awesome icons, consistent with the provided code, to provide visual cues and enhance usability.
- Responsive layout adapting to different screen sizes, ensuring optimal viewing experience on all devices.
- Subtle transitions and loading animations to provide a smooth user experience.
